**Abstract:**
SQL Server is a great database engine. Sadly, it doesn’t provide great maintenance plans or maintenance plans that are DevOps friendly.  During this ignite talk you will see the shortcomings of the default maintenance plans and learn how to fix those shortcomings with Ola Hallengren’s open source maintenance solution.
